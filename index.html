P5- Orinoco - site e commerce - formation DW Openclassrooms.

Ce projet consiste en l'élaboration de la partie frontend d'une application web de vente de produits.
Installation :

  • Vérifier que node et 'npm' sont installés en local sinon le faire
  • Cloner le repo
  • Lancer 'npm install' depuis le dossier du projet
  • lancer le serveur avec 'node server', il devrait démarrer sur 'localhost' avec le port '3000'
Vous pouvez maintenant tester l'application en local !
